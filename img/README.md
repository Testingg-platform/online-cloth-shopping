# Image Library

This folder contains extra collection-ready product images grouped by audience and type:

- `img/men/topwear`
- `img/men/bottomwear`
- `img/men/winterwear`
- `img/women/topwear`
- `img/women/bottomwear`
- `img/women/winterwear`
- `img/kids/topwear`
- `img/kids/bottomwear`
- `img/kids/winterwear`
- `img/beauty`

These files are already formatted to work well as storefront collection cards, so you can reuse them later without searching again.

Primary source families used while curating these image packs:

- `https://fakestoreapi.com`
- `https://dummyjson.com`
- `https://pngimg.com`
